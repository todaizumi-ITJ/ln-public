# 方法3：iMovie で本格編集（高品質・無料30分）

macOS標準のiMovieを使った、プロ品質の動画編集です。

---

## 📋 必要なもの

- ✅ Mac（iMovie標準搭載）
- ✅ マイク（内蔵または外部）
- ✅ スクリーンショット6枚（`demo-screenshots/`）
- ✅ `demo-script.md`（台本）

---

## 🎬 手順

### ステップ1：iMovieを起動（1分）

```bash
open -a iMovie
```

または、アプリケーションフォルダから起動。

---

### ステップ2：新規プロジェクト作成（1分）

1. **プロジェクト** → **ムービー** をクリック

2. テーマを選択：
   - **テーマなし**（推奨）：シンプル
   - または好みのテーマ

3. プロジェクト名を入力：
   ```
   LN Portal Demo
   ```

4. **OK** をクリック

---

### ステップ3：スクリーンショットをインポート（2分）

1. **ファイル** → **メディアを読み込む**

2. `demo-screenshots/` フォルダを選択

3. 6枚のスクリーンショットをすべて選択：
   - `demo-01-portal-top.png`
   - `demo-02-ln-list.png`
   - `demo-03-makeln.png`
   - `demo-04-manual.png`
   - `demo-05-game.png`
   - `demo-06-feedback.png`

4. **すべてを読み込む** をクリック

---

### ステップ4：画像をタイムラインに配置（3分）

1. ブラウザ（上部）に画像が表示される

2. 画像を順番にタイムライン（下部）にドラッグ＆ドロップ

3. 順序を確認：
   ```
   [タイトル黒背景] → [01] → [02] → [03] → [04] → [05] → [06] → [エンディング黒背景]
   ```

4. タイトルとエンディング用の黒背景を追加：
   - **背景** ボタンをクリック
   - 「黒」を選択
   - タイムラインの最初と最後にドラッグ

---

### ステップ5：各画像の表示時間を設定（5分）

各クリップを選択して、右上の **クリップの設定** で時間を調整：

| スライド | 表示時間 | 設定方法 |
|---------|---------|---------|
| タイトル（黒） | 5秒 | クリップを選択 → 時間を5.0に |
| スライド1（トップ） | 20秒 | 時間を20.0に |
| スライド2（LN一覧） | 25秒 | 時間を25.0に |
| スライド3（申立書） | 30秒 | 時間を30.0に |
| スライド4（マニュアル） | 25秒 | 時間を25.0に |
| スライド5（実績） | 25秒 | 時間を25.0に |
| スライド6（フィードバック） | 20秒 | 時間を20.0に |
| エンディング（黒） | 10秒 | 時間を10.0に |

**素早く設定する方法**:
- クリップをクリック
- キーボードの「R」キー
- 時間を入力（例：20）
- Enter

---

### ステップ6：トランジションを追加（2分）

1. **トランジション** ボタンをクリック

2. **クロスディゾルブ** を選択（シンプルで万能）

3. 各クリップの境界にドラッグ

4. トランジションの長さを調整：
   - デフォルト：1.0秒
   - 推奨：0.5秒（短めでテンポよく）

**一括設定**:
- すべてのクリップを選択（⌘ + A）
- **編集** → **トランジションを追加** → **クロスディゾルブ**

---

### ステップ7：タイトルを追加（3分）

#### タイトルスライド

1. タイムラインの最初の黒背景クリップを選択

2. **タイトル** ボタンをクリック

3. **センター** を選択してドラッグ

4. テキストをダブルクリックして編集：
   ```
   LNポータル
   
   発信者情報開示請求・裁判業務管理システム
   
   効率化 × 自動化 × 可視化
   ```

5. フォント・サイズ・色を調整

#### エンディングスライド

1. 最後の黒背景クリップを選択

2. **タイトル** → **センター**

3. テキストを編集：
   ```
   ありがとうございました
   
   LNポータルシステム
   
   ご不明な点があれば、いつでもフィードバックをお寄せください
   ```

---

### ステップ8：ナレーションを録音（10分）

1. タイムラインのスライド1を選択

2. **ウィンドウ** → **ボイスオーバーを録音**（または V キー）

3. マイクを選択（内蔵マイクまたは外部）

4. **録音準備中...** と表示される

5. カウントダウン後、自動的に録音開始

6. `demo-script.md` を読みながら録音

7. スライドが終わったら、次のスライドを選択して続きを録音

**コツ**:
- 各スライドごとに録音（失敗しても部分的に録り直せる）
- マイクとの距離を一定に保つ
- 静かな環境で録音
- 深呼吸してリラックス

**録音後の調整**:
- 音声クリップを選択
- **オーディオ** タブで音量調整
- ノイズリダクション（オプション）

---

### ステップ9：BGMを追加（オプション・2分）

1. **オーディオ** ボタンをクリック

2. **サウンドトラック** を選択

3. 好みのBGMを選択（例：「Documentary」「Corporate」）

4. タイムラインの最初にドラッグ

5. BGMを全体の長さに合わせて調整：
   - クリップの端をドラッグ
   - または **編集** → **オーディオ調整** → **フェードアウト**

6. 音量を調整：
   - BGMクリップを選択
   - **オーディオ** タブ
   - ボリュームを **10-20%** に設定

---

### ステップ10：プレビュー・微調整（2分）

1. **再生**ボタン（スペースキー）で全体を確認

2. 問題があれば修正：
   - 音声が小さい/大きい → 音量調整
   - 画像の表示時間が短い/長い → クリップを調整
   - トランジションが不自然 → 別のトランジションに変更

3. 細かい調整：
   - **拡大表示**（⌘ + +）でタイムラインを拡大
   - フレーム単位で調整

---

### ステップ11：書き出し（2分）

1. 右上の **共有** ボタン（□↑ アイコン）をクリック

2. **ファイル** を選択

3. 設定：
   - **タイトル**: LN Portal Demo
   - **説明**: （任意）
   - **解像度**: 1080p（Full HD）
   - **品質**: 高
   - **圧縮**: より速く（MP4）

4. **次へ** をクリック

5. 保存場所を選択（デスクトップ推奨）

6. **保存** をクリック

7. 書き出し完了まで待つ（約2-5分）

---

## 💡 Tips

### プロ品質にする

#### カラーグレーディング
1. クリップを選択
2. **クリップフィルタとオーディオエフェクト** ボタン
3. **クリップフィルタ** → 好みのフィルタを選択

#### Ken Burns エフェクト（動きを付ける）
1. 画像クリップを選択
2. **クロップ** ボタン
3. **Ken Burns** を選択
4. 開始位置と終了位置を設定

#### 複数のオーディオトラック
- ナレーション（メイン）
- BGM（バックグラウンド）
- 効果音（ポイント強調）

---

### 効率的に編集

#### ショートカットキー

| キー | 機能 |
|------|------|
| スペース | 再生/停止 |
| I | 開始点を設定 |
| O | 終了点を設定 |
| ⌘ + B | クリップを分割 |
| ⌘ + Z | 取り消し |
| V | ボイスオーバー |
| R | スピード調整 |

#### スナップショット
- **編集** → **スナップショット** でプロジェクトの状態を保存
- 複数のバージョンを保存して比較

---

### 失敗しないコツ

- **こまめに保存**（⌘ + S）
- **バックアップ**（プロジェクトファイルをコピー）
- **テスト書き出し**（短い動画で設定を確認）

---

## ⚠️ トラブルシューティング

### 音声と映像がずれる

1. 音声クリップを選択
2. ⌘ + Option + G で **オーディオを無効化**
3. 再度、ボイスオーバーで録音

### ナレーションにノイズが入る

1. 音声クリップを選択
2. **オーディオ** タブ
3. **ノイズリダクション** を有効化
4. **イコライザ** → **音声強調**

### 書き出しが遅い

- 解像度を下げる（720p）
- 品質を「中」に設定
- 他のアプリを閉じる

### ファイルサイズが大きい

- **圧縮**: **より速く** を選択（MP4）
- または、Handbrakeで再圧縮

## 🎥 本格的な動画を作りたい場合

### 推奨方法1：Pictory.ai（AI自動生成）

**URL**: https://pictory.ai/

**メリット**:
- ✅ AIが自動で音声生成
- ✅ スクリプトをコピペするだけ
- ✅ 15分で完成
- ⚠️ 無料版は透かし入り

**手順**:
1. Pictory.aiにログイン
2. 「Script to Video」を選択
3. `demo-script.md` をコピペ
4. `demo-screenshots/` の画像6枚をアップロード
5. 自動生成（5-10分）
6. MP4でダウンロード

---

### 推奨方法2：iMovie（無料・高品質）

**手順**:
1. iMovieを起動
2. スクリーンショット6枚を順番に配置
3. 各スライドの表示時間を設定
4. マイクアイコンでナレーション録音
5. BGM追加（オプション）
6. 書き出し（1080p推奨）

**所要時間**: 約30分

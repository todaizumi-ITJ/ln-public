# 方法1：QuickTime Player で録画（最速10分）

macOS標準のQuickTime Playerを使った、最も簡単な方法です。

---

## 📋 必要なもの

- ✅ Mac（macOS標準機能のみ）
- ✅ マイク（内蔵マイクでOK）
- ✅ `demo-presentation.html`（準備済み）
- ✅ `demo-script.md`（準備済み）

---

## 🎬 手順

### ステップ1：プレゼンテーションを開く（1分）

```bash
cd /Users/todaizumi/Projects/ln-public
open demo-presentation.html
```

または、GitHub Pagesで開く：
```
https://todaizumi-itj.github.io/ln-public/demo-presentation.html
```

ブラウザをフルスクリーンにする（⌘ + Control + F）

---

### ステップ2：台本を別ウィンドウで開く（1分）

```bash
open demo-script.md
```

または、テキストエディタで開いて、別のモニターか画面の端に配置。

---

### ステップ3：QuickTime Playerを起動（1分）

```bash
open -a "QuickTime Player"
```

または、アプリケーションフォルダから起動。

---

### ステップ4：画面収録を開始（2分）

1. QuickTime Playerのメニュー：**ファイル** → **新規画面収録**

2. 録画コントロールが表示される

3. **オプション**（下向き矢印▼）をクリック：
   - **マイク**: 内蔵マイク（または外部マイク）を選択
   - **品質**: 高
   - **ポインタのクリックを表示**: オフ（好みで）

4. **収録**ボタンをクリック

5. 録画範囲を選択：
   - **画面全体**: 画面をクリック
   - **選択した範囲**: ドラッグして範囲を指定

6. 録画開始！

---

### ステップ5：プレゼンテーションを進めながら録音（5分）

**タイミング（目安）**:

| スライド | 時間 | 内容 |
|---------|------|------|
| 1. タイトル | 0:00-0:05 | システム紹介 |
| 2. トップページ | 0:05-0:25 | 主な機能説明 |
| 3. LN一覧 | 0:25-0:50 | 案件管理機能 |
| 4. 申立書作成 | 0:50-1:20 | 自動生成機能 |
| 5. マニュアル | 1:20-1:45 | レベル別業務 |
| 6. 実績・レベル | 1:45-2:10 | ゲーミフィケーション |
| 7. フィードバック | 2:10-2:30 | 改善の仕組み |
| 8. エンディング | 2:30-2:35 | まとめ |

**操作方法**:
- **→ キー**: 次のスライド
- **← キー**: 前のスライド
- **スペースキー**: 次のスライド

**コツ**:
- 落ち着いてゆっくり話す
- 台本を読みながらでOK
- 失敗したら途中でやり直してもOK

---

### ステップ6：録画を停止（1分）

1. メニューバーの⏹ボタンをクリック
2. または、**⌘ + Control + Esc**

---

### ステップ7：保存・編集（オプション）

#### 保存

1. **ファイル** → **保存**
2. ファイル名: `ln-portal-demo.mov`
3. 保存場所: デスクトップまたは任意の場所

#### トリミング（オプション）

1. **編集** → **トリム**
2. 黄色いハンドルをドラッグして、不要な部分をカット
3. **トリム**をクリック

---

### ステップ8：形式変換（オプション）

QuickTimeの`.mov`をMP4に変換する場合：

```bash
cd デスクトップ  # または保存した場所
ffmpeg -i ln-portal-demo.mov -c:v libx264 -c:a aac -b:a 192k ln-portal-demo.mp4
```

---

## 💡 Tips

### 音質を向上させる

- 静かな環境で録音
- 外部マイクを使用（Blue Yeti、Audio-Technica ATR2100など）
- マイクとの距離を一定に保つ

### 失敗を減らす

- 事前に台本を読んで練習
- スライドの切り替えタイミングを確認
- 深呼吸してリラックス

### 再録画のコツ

- 失敗した部分だけ別ファイルで録画
- iMovieで後から組み合わせる

---

## ⚠️ トラブルシューティング

### マイクが認識されない

1. **システム環境設定** → **セキュリティとプライバシー** → **マイク**
2. QuickTime Playerにチェックを入れる

### 画面が小さく録画される

- ブラウザをフルスクリーンにする（⌘ + Control + F）
- または、特定のウィンドウを選択して録画

### 音声が途切れる

- 他のアプリを閉じてメモリを確保
- ストレージ容量を確認

---

## 📊 完成イメージ

**ファイル形式**: MOV（または MP4に変換）
**解像度**: Retinaディスプレイの解像度（例: 2880x1800）
**音声**: モノラルまたはステレオ
**ファイルサイズ**: 約50-200MB（解像度による）

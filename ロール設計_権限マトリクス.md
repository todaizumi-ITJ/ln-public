# ロール設計・権限マトリクス・昇格条件

## 1. ロール概要

```
🏰 Leader ──── 全LNの納品責任者。チーム指揮。全機能アクセス
    │
⚔️ Expert ──── Beginnerを指導。証拠収集の全部 + 書類作成の一部
    │
🛡️ Beginner ── 簡単な証拠チェック。限定された操作のみ
```

## 3. まとめ: 誰が何をやるか

### 🛡️ Beginner がやること

```
証拠を集めるタブの中から:
  ✅ 甲1（著作物USB）のチェック
  ✅ 甲10（KKスクショ）の有無確認
  ✅ 甲11（PL再生試験）の有無確認
  ✅ 甲13（DVDパッケージ）のチェック
  ✅ 委任状・代表者事項証明書・KKのチェック
  ✅ IPNテーブルの閲覧と確認
```

### ⚔️ Expert がやること（↑に加えて）

```
証拠を集めるタブの全部:
  ✅ 全案件共通（権利関係・著作権存在証明）
  ✅ 同一ファイル（甲3〜9, 甲12）のチェック
  ✅ 書類ボックス詰め
  ✅ IPNDBへの遷移

書類作成タブの一部:
  ✅ 証拠説明書の自動生成・PDF・MD出力
  ✅ 別紙一式のチェック操作
  ✅ テンプレートの閲覧・コピー
```

### 🏰 Leader がやること（↑に加えて）

```
書類作成タブの全部:
  ✅ 申立書本文の作成（種類選択・パターン選択・主張入力）
  ✅ 申立書を作成ボタン → プレビュー → PDF/Word/テキスト出力
  ✅ テンプレートの追加
  ✅ 申立準備チェックリストの管理（作るもの側の管理）
```

## 4. 昇格条件

**必要条件（すべて満たすこと）:**

| # | 条件 | 数値 |

| 1 | LNチェック完了数 | 累計 **30件** 以上 |
| 2 | 正確率（差し戻しなし率） | **90%** 以上 |
| 3 | 甲10/甲11 確認完了数 | 累計 **500枝番** 以上 |
| 4 | 連続勤務日数 | **20日** 以上（出勤＋タスク完了） |
| 5 | Expertからの推薦 | **1名** 以上の推薦 |

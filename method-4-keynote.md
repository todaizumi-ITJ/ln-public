# 方法4：Keynote + QuickTime（美しいスライド20分）

Apple標準のKeynoteで美しいプレゼンテーションを作り、QuickTimeで録画します。

---

## 📋 必要なもの

- ✅ Mac（Keynote標準搭載）
- ✅ マイク（内蔵または外部）
- ✅ スクリーンショット6枚
- ✅ `demo-script.md`

---

## 🎬 手順

### ステップ1：Keynoteを起動（1分）

```bash
open -a Keynote
```

または、アプリケーションフォルダから起動。

---

### ステップ2：新規プレゼンテーション作成（2分）

1. **新規書類** をクリック

2. テーマを選択：
   - **ブラック**（推奨）：モダンでプロフェッショナル
   - **グラデーション**: カラフル
   - **白**: シンプル

3. プレゼンテーションサイズ：
   - **ワイド**: 16:9（推奨）
   - 標準: 4:3

---

### ステップ3：タイトルスライド作成（2分）

1. タイトルをクリックして編集：
   ```
   LNポータル
   ```

2. サブタイトル：
   ```
   発信者情報開示請求・裁判業務管理システム
   効率化 × 自動化 × 可視化
   ```

3. フォントとサイズを調整：
   - タイトル：80pt、太字
   - サブタイトル：36pt

4. 中央揃え

---

### ステップ4：スライドを追加（10分）

#### スライド2：トップページ

1. **新規スライド** （+ボタン）

2. レイアウト：**タイトル & 本文**

3. タイトル：
   ```
   📋 トップページ
   ```

4. **挿入** → **選択** → 画像を追加：
   - `demo-01-portal-top.png`

5. 画像を左側に配置（幅の60%）

6. 右側にテキストボックスを追加：
   ```
   ・ホーム：概要とサマリー
   ・LN一覧：案件の管理
   ・マニュアル：業務手順
   ・実績・レベル：進捗の可視化
   
   担当者選択機能
   レベル別権限管理
   ```

#### スライド3：LN一覧

1. 新規スライド

2. タイトル：
   ```
   📄 LN一覧画面
   ```

3. 画像：`demo-02-ln-list.png`

4. テキスト：
   ```
   ・全LN：すべての案件
   ・今日やること：本日対応分
   
   進捗状況の色分け：
   🟢 証拠収集完了
   🟡 作業中
   🔴 未着手
   ```

#### スライド4：申立書作成

2. タイトル：

   ```

3. 画像：`demo-03-makeln.png`

4. テキスト：

   ・著作権証明パターン
   ・主張内容入力
   ・PDF/Markdown出力

   ```

#### スライド5：マニュアル

1. 新規スライド

2. タイトル：

---

### ステップ5：トランジション設定（2分）

1. スライドナビゲータ（左側）で最初のスライドを選択

2. **アニメート** タブをクリック

3. **トランジションを追加**

4. トランジションを選択：
   - **ディゾルブ**（推奨）：スムーズ
   - **マジックムーブ**: ダイナミック
   - **フェード**: シンプル

5. 時間：0.5秒

6. **すべてのスライドに適用** をクリック

---

### ステップ6：アニメーション追加（オプション・3分）

各スライドに簡単なアニメーションを追加：

1. 画像を選択

2. **アニメート** → **ビルドイン**

3. **フェードイン** を選択

4. テキストを選択

5. **ビルドイン** → **フェードとムーブ**

6. **配信**: 段落ごと

---

### ステップ7：プレゼンテーション再生テスト（1分）

1. **再生** ボタン（▶︎）をクリック

2. または **⌘ + Option + P**

3. スライドを進めながら確認（矢印キー）

4. **Esc** キーで終了

---

### ステップ8：QuickTimeで録画（5分）

1. **QuickTime Player** を起動
   ```bash
   open -a "QuickTime Player"
   ```

2. **ファイル** → **新規画面収録**

3. **オプション**（▼）:
   - **マイク**: 内蔵マイク
   - **品質**: 高

4. **収録** をクリック

5. 画面全体を選択（または範囲を指定）

6. Keynoteに戻る

7. **再生** （⌘ + Option + P）

8. `demo-script.md` を読みながらスライドを進める

9. 最後のスライドで少し間を置いてから **Esc**

10. メニューバーの **⏹** で録画停止

---

### ステップ9：保存（1分）

1. **ファイル** → **保存**

2. ファイル名：
   ```
   ln-portal-demo-keynote.mov
   ```

3. 保存場所：デスクトップ

## 💡 Tips

### 美しいスライドにする

#### カラースキーム
- **ダーク**: 背景黒、テキスト白/黄色
- **ライト**: 背景白、テキスト黒/青
- **ブランドカラー**: 会社のロゴカラーを使用

#### フォント
- **タイトル**: 太字、大きめ（60-80pt）
- **本文**: 普通、読みやすいサイズ（24-36pt）
- **統一感**: 2-3種類のフォントに絞る

#### 画像配置
- **2分割**: 画像 | テキスト
- **フルスクリーン**: 画像全体 + テキストオーバーレイ
- **余白**: 詰め込みすぎない

---

### プロフェッショナルな演出

#### Master Slide（マスタースライド）
1. **表示** → **マスタースライドを編集**
2. 共通要素（ロゴ、フッター）を配置
3. すべてのスライドに自動適用

#### ガイドライン
1. **表示** → **ガイドを表示**
2. 要素を揃えて配置

#### グリッド
1. **Keynote** → **環境設定** → **ルーラ**
2. グリッドを有効化

---

### 時間配分のコツ

- 各スライド：20-30秒
- トランジション：0.5秒
- 最初と最後：少し長め（5-10秒）

---

## ⚠️ トラブルシューティング

### スライドが見切れる

- **プレゼンテーションタイプ**: ワイド（16:9）を選択
- 画面解像度を確認

### テキストが小さい

- フォントサイズを24pt以上に
- プレビューで確認

### 録画中に音が途切れる

- 他のアプリを閉じる
- 静かな環境で録音
- 外部マイクを使用

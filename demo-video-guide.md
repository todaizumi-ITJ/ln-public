## 🎬 方法1：HTMLプレゼンテーションを録画（最速）

### 手順

1. **HTMLを開く**
```bash
open demo-presentation.html
```

2. **QuickTimeで画面収録**
```bash
open -a "QuickTime Player"
```
   - `ファイル` → `新規画面収録`
   - マイクをオンにして、`demo-script.md`を読みながら録画

3. **操作方法**
   - 矢印キー（→/←）またはボタンでスライド切り替え
   - スペースキーでも次のスライドへ

4. **保存**
   - 停止して、`ln-portal-demo.mov`として保存

## 🎨 方法3：iMovieで編集（高品質・無料）

### 手順

1. **iMovieを起動**
```bash
open -a iMovie
```

2. **プロジェクト作成**
   - 「ムービー」を選択

3. **スクリーンショットをインポート**
   - スクリーンショット6枚をドラッグ＆ドロップ
   - 順番に並べる

4. **表示時間を調整**
   - スライド1: 20秒
   - スライド2: 25秒
   - スライド3: 30秒
   - スライド4: 25秒
   - スライド5: 25秒
   - スライド6: 20秒

5. **ナレーション録音**
   - タイムライン上部の🎤マイクアイコンをクリック
   - `demo-script.md`を読みながら録音

6. **BGM追加（オプション）**
   - iMovie内蔵のBGMを追加
   - 音量を調整（ナレーションの邪魔にならない程度）

7. **トランジション追加**
   - スライド間に「クロスディゾルブ」を追加

8. **書き出し**
   - `ファイル` → `共有` → `ファイル`
   - 解像度: 1080p
   - 品質: 高
   - `ln-portal-demo.mp4`として保存

## 🚀 方法4：Keynote + QuickTime（Apple標準ツール）

### 手順

1. **Keynoteを起動**
```bash
open -a Keynote
```

2. **新規プレゼンテーション作成**
   - テーマ「ブラック」を選択

3. **スライド追加**
   - スクリーンショットを各スライドに配置
   - 右側にテキストを追加（`demo-script.md`から）

4. **プレゼンテーション再生**
   - `⌘ + Option + P`でプレゼンテーションモード

5. **QuickTimeで録画**
```bash
open -a "QuickTime Player"
```
   - `ファイル` → `新規画面収録`
   - マイクオン
   - Keynoteを再生しながらナレーション

6. **保存**
   - 停止して保存
